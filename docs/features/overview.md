# Features Overview

The Auth Service provides a complete authentication system for your SaaS applications.  
This page serves as a landing page for all the detailed feature walkthroughs.

---

## Feature Categories

### User Registration
Manage user sign-ups, password validation, and audit logging.  
- **[Registration](registration.md)** – Create users, enforce password policies, and record audit events.

### Sessions
Handle login, logout, token issuance, refresh tokens, and session revocation.  
- **[Sessions](session.md)** – Login with or without MFA, refresh access tokens, logout, and manage active sessions.

### Multi-Factor Authentication (MFA)
Support TOTP-based MFA, backup codes, and step-up authentication challenges.  
- **[MFA](mfa.md)** – Enroll and confirm MFA methods, generate backup codes, and complete login with MFA verification.
